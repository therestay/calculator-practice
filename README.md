# Calculator

A tiny calculator library used to demonstrate basic arithmetic operations.

## Features

- Add two numbers
- Subtract two numbers
- Multiply two numbers

## Usage

```python
from calculator import add, subtract, multiply

add(2, 3)        # 5
subtract(5, 2)    # 3
multiply(4, 3)    # 12
```

## Installation

Clone this repositroy and run the tests to verify everything works:

```bash
python -m pytest
```
